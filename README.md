# Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, projects, and contact information as a web developer and designer.

## 🌟 Features
- **Responsive Design**: Fully responsive for all devices.
- **Smooth Navigation**: Includes smooth scrolling and active link highlighting.
- **Hero Section**: A visually engaging introduction with a call-to-action button.
- **About Me Section**: Highlights my skills, interests, and a short bio.
- **Portfolio Section**: Displays my projects with links to their respective GitHub repositories.
- **Contact Section**: A contact form for visitors to send me messages.
- **Social Links**: Interactive hover effects on social media icons.

## 🖥️ Tech Stack
- **HTML5**: Markup language for the structure of the site.
- **CSS3**: Styles and responsive design.
- **JavaScript**: Smooth scrolling, form validation, and interactivity.
- **Google Fonts**: `Poppins` for a clean and modern typography.
- **GitHub**: For version control and hosting the project.
